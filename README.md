### how it is build

This web page is build using only **html javaScript and CSS**. No framework has been used because the sole purpose of this site is to publicize my portfolio.
Even so, **I have tried to develop it based on good labeling practices**
- *Js code structure* 
- *A good use of CSS*
  - variables
  - selectors, 
  - media queries, 
  - comments
  - low specificity
  - and more ...
