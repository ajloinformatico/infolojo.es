Keepit is a note program developed with Wokis. It's that simple. It was developed as a final project for my development degree. You can create, view and delete notes. Also, you can search any note by title, category or label. It is entirely on Spanish, but it could be translated easily to other language. It accepted multi-user and used a MySQL database to store all notes.
